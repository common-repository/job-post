== Description ==
	Creating job for WordPress made easy by “Job Post” plugin. super lightweight plugin allows you to create job post, admin can submit job and employee can apply for job.
It can work with any theme and is really simple to setup. Based on custom post type and template  to made easy to displaying job-post list and applay form.


After you upload and activate the plugin the comments forms on your site will look like this:
[youtube https://www.youtube.com/watch?v=mGO46xle6Rc]

== Installation ==

The manual installation method involves downloading the plugin and uploading it to your web server via your favorite FTP application.

	* Download the plugin file to your computer and unzip it
	* Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
	* Activate the plugin from the Plugins menu within the WordPress admin
	* Add 'Jop Post' custom post type and add template for display list of job post with category wise, and second template for applay form.

== How to Work ==
	
	-> Add your job post categories in jop post categories.
	-> Add your employers job post using 'Add New' in jop post.
	-> Select job post category.
	-> After create a job post listing page in pages and select 'RCT Job Post List' template.
	-> and also create a new page for candidate application form and select 'RCT Job Post Application Form' template.
	
	-> view Job post listing page that have list of post in that category and applay for that posting.
	
	-> Applications page list of candidate which have applay for your job post.
	-> If you want to sent mail to job post admin when candidate appply job then you have configure smtp setting.
